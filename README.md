# 💙 I’m Sorry – A Heartfelt Apology Website

A simple yet emotional and interactive website designed to express feelings that are often hard to say in words.

This project is a storytelling-based apology website that combines design, animation, and music to create a meaningful user experience.

---

## ✨ Features

- 💌 Emotional storytelling through multiple sections  
- 🎵 Background music with mute/unmute control  
- ✨ Smooth transitions and animations  
- 🌙 Floating emojis and aesthetic UI  
- ⭐ Animated starry background  
- 📱 Fully responsive design (mobile-friendly)  

---

## 💡 Purpose

This website is created to help express an apology in a creative and memorable way.  
Sometimes words alone are not enough — this experience aims to make emotions feel real and impactful.
